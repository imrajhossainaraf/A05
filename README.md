1️ What is the difference between var, let, and const?

var, let, and const are used to declare variables in JavaScript.

The main difference is how they handle scope and reassignment.

var is the old way of declaring variables. It is function-scoped and can be redeclared and reassigned.

let is block-scoped and can be reassigned, but cannot be redeclared in the same scope.

const is also block-scoped, but it cannot be reassigned after it is declared.

3️ What is the difference between map(), filter(), and forEach()?

These are JavaScript array methods used to work with array data.

forEach() is used to loop through an array and perform an action for each element.

map() is used to create a new array by transforming each element.

filter() is used to create a new array that contains only elements that match a condition.


4️ What is an arrow function?

An arrow function is a shorter way to write functions in JavaScript.
It was introduced in ES6 and uses the => syntax.


5️5 What are template literals?

Template literals are a way to write strings in JavaScript using backticks ( ) instead of quotes.

They allow us to insert variables inside strings using ${}.
